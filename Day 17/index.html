Selectors

A CSS selector is the first part of a CSS Rule. It is a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them

Types Of Selector

  1. Elements Selector
  2. CLass Selector
  3. Id Selector
  4. Child Selector
  5. Descendant Selector

Elements Selector

An element selector (also called a type selector) in CSS is the simplest kind of selector.

Class Selector

A class selector in CSS targets HTML elements that have a specific class attribute.It’s written using a dot (.) followed by the class name.

Id Selector

An ID selector in CSS is used to style a single, unique element that has a specific id attribute.It’s written using a hash (#) followed by the ID name.

Child Selector

A child selector in CSS is used to select elements that are direct children of a specific parent element.It is written with a greater-than sign (>) between the parent and child selectors.

Descendant Selector

A descendant selector in CSS is used to select elements that are anywhere inside a specific parent element

Pseudo Selector

A pseudo-selector in CSS is a keyword added to a selector that lets you style elements based on:
  • ✅ their state (like being hovered or focused)
  • ✅ specific parts of elements (like the first letter, or first line). They always start with a colon (:) or double colon (::).
